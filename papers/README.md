# Papers 文件夹使用指南

## 文件结构

```
papers/
├── README.md              # 本文件：使用指南
├── index.md              # ⭐ 主要：论文索引目录（48篇）
├── summaries.md          # ⭐⭐ 最详细：论文详细摘要
└── collection_progress.md # 进度跟踪：收集状态和下一步
```

---

## 什么时候看哪个文件？

### 1. **index.md** - 论文索引
**用途：** 快速浏览所有论文，按方向分类

**什么时候看：**
- 想知道"有哪些论文"时
- 想按方向浏览论文时
- 想看论文状态（已找到/待搜索）时

**包含内容：**
- 48篇论文，按8个categories组织
- 每篇论文的：标题、作者、year、venue、方向、状态
- Top 10优先阅读列表

---

### 2. **summaries.md** - 详细摘要
**用途：** 深入理解某篇论文

**什么时候看：**
- 想深入了解某篇论文时
- 想学习论文的方法时
- 想知道论文与我们的相关性时
- 准备写Related Work时

**包含内容：**
- 已总结论文的详细摘要
- 每篇包含：摘要、贡献、方法、结果、局限性、与我们的相关性
- 个人笔记和问题

---

### 3. **collection_progress.md** - 进度跟踪
**用途：** 了解收集进度和下一步

**什么时候看：**
- 想知道"收集到哪了"时
- 想看下一步搜索什么时
- 想更新收集状态时

**包含内容：**
- 统计数据（总数、已找到、已总结）
- 按category的收集状态
- 下一步搜索查询列表
- 下载的论文列表

## 工作流程

### 1. 浏览阶段
```
开始 → 看 index.md → 找感兴趣的论文 → 去 summaries.md 看详细
```

### 2. 阅读阶段
```
读论文 → 更新 summaries.md 添加摘要 → 更新 index.md 标记"已读"
```

### 3. 收集阶段
```
查 collection_progress.md 看下一步 → 搜索论文 → 下载 → 更新 collection_progress.md
```

---

## 文件更新规则

### index.md
- 添加新论文时更新
- 论文状态变化时更新（找到→已读→已总结）

### summaries.md
- 读完论文后添加摘要
- 可以多次补充（读完第一次后，有新想法再补充）

### collection_progress.md
- 每次搜索后更新统计
- 添加新找到的论文URL
