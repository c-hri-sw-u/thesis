## 🚨 Critical Risks

| Risk | Mitigation |
|------|------------|
| System bugs during experiment | Start testing NOW, have fallback plan |
| Experiment needs >7 days | This breaks timeline - 7 days is maximum |
| Advisor feedback delays | Send drafts early (Mar 9, Mar 19) |
| Analysis takes longer | Pre-write analysis code before experiment |
| System fails completely | Have manual data collection backup |

## 📝 Minimum Viable Thesis

**Must Have:**
- [x] Clear research direction
- [ ] Clear research questions
- [ ] Working system (demonstrable)
- [ ] 5-7 days of experimental data
- [ ] Basic analysis showing something interesting
- [ ] All thesis sections written
- [ ] Defense presentation

**Can Sacrifice:**
- Perfect polish (can revise post-defense)
- Long experiment duration (5-7 days is enough for n=1)
- Extensive related work (shorter lit review OK)
- Complex analysis (simple is fine for n=1)

**Cannot Sacrifice:**
- Working system (non-negotiable)
- Any experimental data (need something to analyze)
- Core thesis sections (must have all)
- Defense readiness (must be prepared)
